Experiments

Tools

  • libgen downloader is a simple tool that was built using React, Ink and Zustand to search and download ebooks from libgen via command line interface. [ web page / github / npm ]
  • xeh is a command-line Canonical HEX+ASCII display tool written in ~100 lines of Common Lisp. [ github ]
  • console8 is yet another CHIP8 emulator but works on your command line. console8 simply uses your command line to render the graphics and does not require any GUI. [ github / npm ]
  • kmap-solver is a command line tool to solve and visualize Karnaugh Maps up to 4 variables. [ github / npm ]
  • lurkdown is a Markdown transpiler that was built on NodeJS with typescript to create standalone HTML files. [ github / npm ]

Libraries

  • dungrain is a JavaScript library that is using Binary Space Partitioning to create procedurally generated dungeons. [ github / npm ]
  • kmap-solver-lib is a utility function that is using in kmap-solver to solve Karnaugh Maps up to 4 variables. [ github / npm ]
